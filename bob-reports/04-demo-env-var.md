## Why Static Analyzers Miss This

1. **No Type Safety**: `process.env` returns `string | undefined` - TypeScript can't validate
2. **Runtime Values**: Environment variables are set at runtime, not compile time
3. **External Configuration**: .env and docker-compose.yml are not analyzed by TypeScript
4. **Cross-File Dependencies**: Relationship between code and config files is implicit
5. **Deployment Configs**: Production environment variables are outside the codebase

// config-validator.ts
export function validateConfig() {
